== Fine-tune vs RAG ==

                        metric  fine_tune  rag
     CTA (есть мягкая продажа)        0.0  0.8
   Структура (списки/переносы)        0.8  0.0

                Суммарный балл        1.8  1.8

Ничья по сумме: рекомендуем **комбинировать** — FT для стиля/CTA, RAG для свежих фактов.
